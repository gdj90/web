선택자란?

특정 html요소를 선택할때 사용하는 기능 선택자로 해당요소(태그)를 선택하여 원하는 스타일 또는 기능을 적용

1. 모든(전체) 선택자 : *

        * {
            스타일속성 : 값;
        }

2. 태그 선택자 : 태그명

        태그명 {
            스타일속성 : 값;
        }

3. 아이디 선택자 : #아이디명

현재 문서의 특정 html요소 중 딱 한개만 선택하고자 할때 사용
단 id 속성은 고유한 아이디 값을 부여해야함 

        #아이디명 {
            스타일속성 : 값;
        }
  1. 아이디선택자 테스트
  2. 아이디선택자
  3. 아이디선택자
  4. 아이디선택자

4. 클래스 선택자 : .클래스명

해당 문서내에 원하는 요소 여러개를 함께 선택하고자 할때 사용 

        .클래스명 {
            스타일속성 : 값;
        }
