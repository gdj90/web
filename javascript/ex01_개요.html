자바 스크립트 개요

HTML페이지의 동작을 사용자의 요구에 맞도록 수행 기능
브라우저의 언어임.
인터프리터 방식의 언어
//HTML 영역에서는 주석이 아님. 자바스크립트영역에서만 주석임

자바스크립트의 특징

  • 접근제한자 개념없음.
  • 변수 선언이 필요없음. 초기화시 자료형 결정
  • 웹 환경에 특화된 언어 => 보안등의 이유로 기능 제한됨
  • 보안에 취약함 : 소스가 외부 공개됨

사용법

  • 인라인(inline) : 태그내에 간단히 소스코드를 추가
  • 내부(internal): html 파일 내에 script 태그로 소스코드 추가
  • 외부(external) : 별도의 .js 파일로 소스코드를 추가.

인라인방식

<태그 on이벤트명="자바스크립트소스()">
<a href="javascript:자바스크립트소스">실행</a>


링크태그에서 알림창출력 링크태그에서 콘솔출력

내부방식(internal)

html 페이지내에 script 태그 영역안에 자바스크립트 소스 작성
script 태그의 위치는 head,body내에서든 어디든지 가능함
